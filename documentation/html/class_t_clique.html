Detailed Description

Class representing a clique in our model. A clique is defined as having a set of nodes that are all leaves in all dimensions except one. Each clique has a set of matrices, the change rate parameters, and the start index of the nodes in the tree. The start index, the variable dimension, and the number of nodes are needed to get the correct indices in our multidimensional space Y and Z.
