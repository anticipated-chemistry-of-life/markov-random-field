\doxysubsection{Detailed Description}
TStorage Y is the class to store a single value Y of the Random Markov Field We store in a single 64 bits integer (8 bytes) the following information\+:
\begin{DoxyItemize}
\item the first 16 bits are the counter of the number of times the element was a one in the MCMC
\item the 17th (position 16 ) is the current state of the element (0 or 1)
\item the rest is the position of the element Y in the multidimensional linearized Y vector of leaves 
\end{DoxyItemize}
